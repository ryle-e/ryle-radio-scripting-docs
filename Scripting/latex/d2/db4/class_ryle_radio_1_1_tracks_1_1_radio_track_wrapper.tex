\doxysubsection{Detailed Description}
A wrapper class for \doxylink{class_ryle_radio_1_1_tracks_1_1_radio_track}{Radio\+Track} so that track types can be switched between in the inspector! Also contains various values that are track-\/agnostic. 

This is how a \doxylink{class_ryle_radio_1_1_tracks_1_1_radio_track}{Radio\+Track} is stored and accessed in \doxylink{class_ryle_radio_1_1_radio_data}{Radio\+Data}. ~\newline
If we didn\textquotesingle{}t use a wrapper like this, you wouldn\textquotesingle{}t be able to choose \doxylink{class_ryle_radio_1_1_tracks_1_1_radio_track_wrapper_ad02b3c8a6fd28d31d8d728ec8f583e42}{track\+Type} in a dropdown and see it change in the inspector-\/ it\textquotesingle{}s not possible (to my knowledge) to do that without some kind of wrapper and {\ttfamily \mbox{[}Serialize\+Reference\mbox{]}}. ~\newline
Wrappers also contain variables that exist for every track event\+Type, such as \doxylink{class_ryle_radio_1_1_tracks_1_1_radio_track_wrapper_a778c05ac1056316fc9224e13c80c340c}{range} and \doxylink{class_ryle_radio_1_1_tracks_1_1_radio_track_wrapper_ac26cf86217a7fdab6269f90e09f3d590}{gain}.

Animation\+Curve Ryle\+Radio.\+Tracks.\+Radio\+Track\+Wrapper.\+range\+Curve = new(Default\+Range\+Curve.\+keys)}



The curve defining the loudness of the track over its range. The progress between {\ttfamily range.\+x} and {\ttfamily range.\+y} the Radio\+Output.\+Tune value is, is the progress along this curve the tune power is. 


\begin{DoxyItemize}
\item If the curve is the default curve (smooth from 0 to 1 to 0), it will smoothly get louder towards the center of the range, and quieter towards the edge. 
\item If the curve is a flat line at y=1, it will be the same volume across the entire range 
\item If the curve is a line from 0 -\/ 1, it will be louder the further along the range the tune is, getting loudest at {\ttfamily range.\+y} 
\item If the curve is goes up and down repeatedly, it will be at various different volumes depending on what you set, moving between them along the range 
\end{DoxyItemize}
